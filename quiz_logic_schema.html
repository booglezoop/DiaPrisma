Results page — primary risk driver (first match wins)
Tier 1
Имотът привлича огледи, но не и оферти
Condition: on-market + viewings = "няколко" or "много" → price / presentation problem
Tier 2
Обявата не достига до правилните купувачи
Condition: on-market + 0 offers + viewings = "Не" → marketing / positioning problem
Tier 3
Цената на имота носи риск
Condition: price set by personal need, friend, or "Още не съм мислил" — either track
Tier 4
Документацията крие потенциални рискове
Condition: pre-market + docs ≠ "Да, всичко е наред" — concrete, scary, actionable
Tier 5
Имотът има потенциал — но планът липсва
Default: pre-market, no flags triggered — education opportunity
Secondary signal: one supporting sentence shown beneath primary message, drawn from the next applicable tier (mispricing → docs → fast-sale urgency). Never a full second block.
